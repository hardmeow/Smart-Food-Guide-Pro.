# 智食鉴 (SmartFood Scanner) 设计方案

## 选中方案：现代健康科技风格

### 设计运动
**现代极简主义 + 健康科技感**：融合医疗/健康应用的专业感与现代科技产品的简洁性。参考 Apple Health、Google Fit 的设计语言。

### 核心原则
1. **以数据为中心**：健康评分是绝对焦点，采用大号仪表盘展示
2. **信息层级清晰**：关键信息优先展示，详细数据可展开查看
3. **交互即反馈**：每个操作都有明确的视觉反馈，建立用户信心
4. **绿色生命力**：绿色代表健康，贯穿整个界面的关键交互点

### 色彩哲学
- **主色**：翡翠绿 (`#10B981`) - 代表健康、生机、安全
- **辅助色**：琥珀色 (`#F59E0B`) - 警告/需要注意的成分
- **危险色**：红色 (`#EF4444`) - 高风险成分
- **中立色**：灰色系 (`#6B7280` - `#F3F4F6`) - 背景和文字
- **背景**：纯白 (`#FFFFFF`) + 浅灰 (`#F9FAFB`) - 清爽、医疗感

### 布局范式
- **垂直流式布局**：从上到下的自然流程（拍照 → 预览 → 分析 → 结果）
- **卡片式分组**：将配料、营养、建议分离为独立卡片，便于扫一眼获取信息
- **底部操作栏**：主要操作按钮固定在底部，便于单手操作
- **非对称设计**：避免完全居中，左对齐的文本配合右侧的数据可视化

### 签名元素
1. **健康评分仪表盘**：大圆形进度条，显示 0-100 的健康分数
2. **成分风险标签**：彩色标签系统（绿/黄/红），快速识别风险等级
3. **渐进式信息展示**：关键数据先显示，详情可点击展开

### 交互哲学
- **即时反馈**：上传后立即显示识别进度
- **渐进式加载**：先显示快速结果，后续 AI 分析在后台进行
- **触觉反馈**：按钮点击、卡片展开都有微妙的过渡动画
- **鼓励性设计**：健康评分高时显示鼓励信息，低时显示建议

### 动画指南
- **入场动画**：卡片从下方滑入，评分仪表盘数字滚动
- **交互动画**：按钮 hover 时轻微放大 (1.05x) + 阴影加深
- **加载动画**：脉动的绿色圆形，代表扫描中
- **过渡动画**：所有状态变化都用 300-500ms 的 ease-out 过渡

### 排版系统
- **标题字体**：系统字体 (San Francisco / Segoe UI) 的 700 weight
- **正文字体**：系统字体的 400-500 weight
- **字号层级**：
  - 大标题：28px (评分数字)
  - 小标题：18px (卡片标题)
  - 正文：14px (描述文本)
  - 辅助：12px (标签、提示)

### 响应式设计
- **手机竖屏**（主要场景）：全宽布局，按钮占满宽度
- **平板/横屏**：最大宽度 600px，居中显示
- **桌面**：卡片式布局，最大宽度 800px

## 实现细节

### 色彩代码
```
主绿色: #10B981 (rgb(16, 185, 129))
浅绿色: #ECFDF5 (rgb(236, 253, 245))
警告黄: #F59E0B (rgb(245, 158, 11))
浅黄色: #FFFBEB (rgb(255, 251, 235))
危险红: #EF4444 (rgb(239, 68, 68))
浅红色: #FEE2E2 (rgb(254, 226, 226))
灰色: #6B7280 (rgb(107, 114, 128))
浅灰: #F9FAFB (rgb(249, 250, 251))
```

### 关键交互流程
1. **首屏**：大相机按钮 + 上传区域
2. **识别中**：脉动加载动画 + 进度提示
3. **结果页**：
   - 顶部：食品图片缩略图
   - 中部：大评分仪表盘
   - 下部：配料卡片 + 营养卡片 + 建议卡片
4. **详情展开**：点击卡片展开详细信息
