# 🚀 快速开始指南

## 最简单的方式：直接打开文件

### Windows
1. 找到 `smartfood-scanner.html` 文件
2. 右键点击 → 打开方式 → 选择浏览器（Chrome、Firefox 等）
3. 完成！应用已启动

### macOS
1. 找到 `smartfood-scanner.html` 文件
2. 右键点击 → 打开方式 → 选择浏览器
3. 完成！应用已启动

### Linux
```bash
# 使用默认浏览器打开
xdg-open smartfood-scanner.html

# 或使用特定浏览器
firefox smartfood-scanner.html
# 或
google-chrome smartfood-scanner.html
```

## 首次使用注意事项

⚠️ **第一次使用时，应用会下载 OCR 库（约 50MB）**

- 请确保网络连接良好
- 等待加载完成（通常 1-2 分钟）
- 之后会被浏览器缓存，后续使用会更快

## 基本操作

### 1️⃣ 上传图片
```
点击 "📸 点击上传或拍照" 区域
  ↓
选择食品包装图片或拍照
  ↓
点击 "开始分析"
```

### 2️⃣ 查看结果
```
等待分析完成（通常 10-30 秒）
  ↓
查看健康评分（大圆形仪表盘）
  ↓
点击卡片展开详细信息
```

### 3️⃣ 重新分析
```
点击 "🔄 重新扫描"
  ↓
选择另一张图片
  ↓
重复分析
```

## 配置 DeepSeek API（可选）

如果想要更好的未知配料分析效果：

### 步骤 1：获取 API Key
1. 访问 [DeepSeek 官网](https://www.deepseek.com/)
2. 注册账户
3. 创建 API Key
4. 复制 Key（格式：`sk-xxxxxxxx`）

### 步骤 2：在应用中配置
1. 打开应用
2. 刷新页面（F5）
3. 如果看到 API Key 设置提示，输入 Key
4. 或在浏览器控制台中运行：
```javascript
localStorage.setItem('deepseek_api_key', 'sk-your-api-key-here');
```

### 步骤 3：验证配置
1. 重新加载应用
2. 上传一张包含未知配料的食品图片
3. 应用会自动调用 API 分析

## 常见问题

### ❓ 应用无法打开？
- 确保使用现代浏览器（Chrome、Firefox、Safari、Edge）
- 尝试刷新页面（Ctrl+R 或 Cmd+R）
- 清除浏览器缓存后重试

### ❓ OCR 识别失败？
- 确保图片清晰、光线充足
- 配料表和营养成分表要完整可见
- 文字要水平放置（不要倾斜）
- 尝试调整图片角度或距离

### ❓ 分析速度很慢？
- 第一次使用需要下载 Tesseract.js 库
- 之后会被缓存，速度会快很多
- 如果仍然很慢，检查网络连接

### ❓ 如何删除保存的 API Key？
```javascript
// 在浏览器控制台中运行
localStorage.removeItem('deepseek_api_key');
```

## 💡 使用技巧

### 获得最佳识别效果
1. 📸 拍照时确保光线充足
2. 📐 保持相机与包装垂直
3. 🔍 让配料表和营养表都在画面内
4. 🎯 避免反光和阴影

### 快速分析多个产品
1. 分析第一个产品
2. 点击 "🔄 重新扫描"
3. 立即上传下一张图片
4. 重复操作

### 保存分析结果
- 使用浏览器的截图功能（Shift+Ctrl+S）
- 或右键保存页面
- 或使用浏览器的打印功能（Ctrl+P）保存为 PDF

## 🎓 理解评分

### 健康评分含义
- **80-100 分**：优秀 ✅ 可以放心食用
- **60-79 分**：良好 👍 可以适量食用
- **40-59 分**：一般 ⚠️ 建议适量减少
- **20-39 分**：较差 ⚠️ 不建议频繁食用
- **0-19 分**：很差 ❌ 建议避免食用

### 配料风险标记
- **✓ 绿色**：安全配料
- **⚠️ 黄色**：需要注意的配料
- **✕ 红色**：高风险配料
- **❓ 灰色**：未知配料（需要 API 分析）

## 🔒 隐私保护

✅ **完全离线处理**
